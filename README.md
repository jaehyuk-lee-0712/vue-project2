# POPVIE
영화 정보 제공 웹

## 주요 기능
영화 상세 정보 열람.   
원하는 영화 검색.   

## 사용 기술
Npm (Package 관리)
Vue3
TypeScript   
Swiper    
oh-vue-icons    
axios   

## 개요
이 프로젝트는 Vue 3, Vite, TypeScript를 사용하여 구축된 영화 목록 및 상세 정보 애플리케이션입니다. 이 애플리케이션은 The Movie Database (TMDb) API에서 데이터를 가져와 현재 상영 중인 영화, 최고 평점 영화, 개봉 예정 영화를 표시합니다. 사용자는 영화를 검색하고 각 영화에 대한 자세한 정보를 볼 수 있습니다.   

## 트러블 슈팅
